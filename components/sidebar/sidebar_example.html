BsSidebar Web Component Demo

Examples of the <bs-sidebar> component, which provides a responsive offcanvas navigation sidebar.

1. Basic Sidebar Navbar

A simple sidebar that remains an offcanvas drawer at all breakpoints.

Brand Home Features Pricing <bs-sidebar title="Sidebar Title" background="body-tertiary"> <bs-navbar-brand href="#">Brand</bs-navbar-brand> <bs-navbar-nav> <bs-navbar-link href="#" active>Home</bs-navbar-link> <bs-navbar-link href="#">Features</bs-navbar-link> <bs-navbar-link href="#">Pricing</bs-navbar-link> </bs-navbar-nav> </bs-sidebar>

2. Responsive Sidebar

This sidebar expands into a horizontal navbar at the lg breakpoint. Below lg, it behaves as an offcanvas drawer.

Responsive Brand Home About Contact <bs-sidebar title="Menu" expand="lg" background="light"> <bs-navbar-brand href="#">Responsive Brand</bs-navbar-brand> <bs-navbar-nav> <bs-navbar-link href="#" active>Home</bs-navbar-link> <bs-navbar-link href="#">About</bs-navbar-link> <bs-navbar-link href="#">Contact</bs-navbar-link> </bs-navbar-nav> </bs-sidebar>

Using theme="dark", background="dark", and placement="end".

5. Permanent Sidebar (Responsive)

A sidebar that is visible on desktop (md and up) but collapses into an offcanvas drawer on mobile, with a toggle button at the top.

Dashboard Content

The permanent sidebar is displayed alongside the main content. It doesn't collapse into an offcanvas drawer and uses a fixed width.

You can use bs-navbar-nav and other components inside it just like with the responsive sidebar.
