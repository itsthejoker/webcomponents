BsCodeBlock Web Component Demo

Examples of the <bs-code-block> custom element.

1. Basic Usage (HTML)

By default, it uses HTML highlighting.

<div class="alert alert-primary"> Hello World! </div>

2. JavaScript Highlighting

Uses the language="javascript" attribute.

function hello() { console.log("Hello, world!"); } hello();

3. With Bootstrap Spacing Classes

The component passes through classes to its internal wrapper. Note the copy button in the top right corner.

<bs-card title="Shadowed Card"> <p>Content here</p> </bs-card>
