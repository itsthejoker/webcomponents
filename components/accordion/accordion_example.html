BsAccordion Web Component Demo

Examples of the <bs-accordion> and <bs-accordion-item> custom elements.

1. Basic Accordion

A standard accordion where only one item can be open at a time.

This is the first item's body. It is shown by default, because of the expanded attribute. This is the second item's body. Opening this will close the first one. This is the third item's body.

2. Flush Accordion

Uses the flush attribute to remove borders and rounded corners.

Placeholder content for the first flush item. Placeholder content for the second flush item.

3. Always Open Accordion

Uses the always-open attribute so items stay open when others are toggled.

This item is open and will stay open even if you open the next one. Go ahead, open me! The first one won't close.

4. Custom Header Slot

Uses the slot="header" to provide custom HTML for the accordion header.

Custom Header with Icon You can put any HTML in the header using the slot="header". This one uses the simple title attribute.
