CenteredColumn Web Component Demo

Examples of the <centered-column> component, which provides an easy way to center content using Bootstrap's grid system.

1. Default Centered Column

By default, the column is 8 units wide (col-8) and centered.

This is a default centered column (8 units wide).
<centered-column> <div class="demo-box text-center"> This is a default centered column (8 units wide). </div> </centered-column>

2. Custom Width

You can specify the width using standard Bootstrap column classes like col-4.

This column is only 4 units wide.
<centered-column class="col-4"> <div class="demo-box text-center"> This column is only 4 units wide. </div> </centered-column>

3. Responsive Widths

The component supports responsive column classes (e.g., col-10 col-md-8 col-lg-6).

Responsive Column

10 units on small screens,
8 units on medium screens,
6 units on large screens and above.

<centered-column class="col-10 col-md-8 col-lg-6"> <div class="demo-box"> <h5 class="text-center">Responsive Column</h5> <p class="mb-0 text-center"> 10 units on small screens,<br> 8 units on medium screens,<br> 6 units on large screens and above. </p> </div> </centered-column>

4. With Spacing Utilities

Other Bootstrap classes like mt-4 or p-3 are kept on the component wrapper.

Centered column with margin top and extra styling on the wrapper itself.
<centered-column class="col-8 mt-4 border p-3 rounded bg-light-subtle"> <div class="text-center"> Centered column with margin top and extra styling on the wrapper itself. </div> </centered-column>

5. Nested Content

The component handles any HTML content inside it.

Featured
Special title treatment

With supporting text below as a natural lead-in to additional content.

Go somewhere
<centered-column class="col-9"> <div class="card"> <div class="card-header">Featured</div> <div class="card-body"> <h5 class="card-title">Special title treatment</h5> <p class="card-text">With supporting text below as a natural lead-in to additional content.</p> <a href="#" class="btn btn-primary">Go somewhere</a> </div> </div> </centered-column>
