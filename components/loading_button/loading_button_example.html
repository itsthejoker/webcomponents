Loading Button Web Component Demo

Examples of the <loading-button> custom element.

1. Standard Variants

Uses the variant attribute to apply Bootstrap button classes.

Primary Secondary Success Danger Warning Info Light Dark Link

Click any button to see the loading state. Note: these will stay in loading state unless programmatically re-enabled or the page is reloaded.

2. Outline Variants

You can also use outline variants by specifying them in the variant attribute.

Outline Primary Outline Secondary Outline Success Outline Danger

3. Custom Onclick & Programmatic Reset

This example uses a custom onclick handler and re-enables the button after 3 seconds.

Click to start 3s timer

4. Form Submission

By default, the button is type="submit". You can use it within a form.

Submit Form (Will reset after 2s)

5. Sizes & Custom Classes

Custom classes are passed through to the internal button.

Large Button Default Button Small Button
Block Level Button (w-100)

6. Using Value Attribute

If no inner HTML is provided, it uses the value attribute.

7. Attribute Pass-through (e.g., href)

Attributes like href or data-* are passed through to the internal button. Note: Standard <button> elements ignore href by default, but it's useful for integration with other libraries or custom scripts.

Link-style button with href
