BsToast Web Component Demo

A custom web component wrapper for Bootstrap Toasts.

Basic Toast

A simple toast with a title and message. Click the button to show it.

Hello, world! This is a toast message.
<button type="button" class="btn btn-primary" onclick="document.getElementById('basicToast').show()">Show Toast</button> <bs-toast id="basicToast" title="Bootstrap" time="11 mins ago"> Hello, world! This is a toast message. </bs-toast>

Live Example (Auto-show)

Use the show attribute to display the toast immediately on load.

This toast appeared automatically.
<bs-toast title="Notification" time="just now" show> This toast appeared automatically. </bs-toast>

Stacking Toasts

Toasts stack automatically when placed inside a .toast-container.

First message in the stack. Second message in the stack.
<div class="toast-container position-absolute top-0 end-0 p-3"> <bs-toast title="Message 1" time="just now" show> First message in the stack. </bs-toast> <bs-toast title="Message 2" time="2 seconds ago" show> Second message in the stack. </bs-toast> </div>

Custom Content

Toasts can be simplified by omitting the header or adding custom slots. The component automatically handles the side-aligned close button for simple text-only toasts.

Hello, world! This is a toast message with automatic flex layout and close button.
Hello, world! This is a toast message with manual body and custom buttons.

Color Schemes

Use the variant attribute to apply Bootstrap contextual colors.

This is a primary colored toast. This is a success colored toast. This is a danger colored toast.
<bs-toast variant="primary" show> This is a primary colored toast. </bs-toast> <bs-toast variant="success" show> This is a success colored toast. </bs-toast> <bs-toast variant="danger" show> This is a danger colored toast. </bs-toast>

Placement

Use Bootstrap's positioning utilities on a .toast-container to place toasts anywhere.

I'm right in the middle!
<div class="toast-container position-absolute top-50 start-50 translate-middle p-3"> <bs-toast title="Centered" show autohide="false"> I'm right in the middle! </bs-toast> </div>
