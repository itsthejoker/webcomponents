BsBadge Web Component Demo

A custom web component for Bootstrap 5 badges.

Basic Badges

Badges using the variant attribute or default slot content.

Primary Secondary Success Danger Warning Info Light Dark
<div class="badge-container"> <bs-badge variant="primary">Primary</bs-badge> <bs-badge variant="secondary">Secondary</bs-badge> <bs-badge variant="success">Success</bs-badge> <bs-badge variant="danger">Danger</bs-badge> <bs-badge variant="warning">Warning</bs-badge> <bs-badge variant="info">Info</bs-badge> <bs-badge variant="light">Light</bs-badge> <bs-badge variant="dark">Dark</bs-badge> </div>

Pill Badges

Use the pill attribute to create rounded-pill badges.

Primary Pill Secondary Pill Success Pill
<div class="badge-container"> <bs-badge variant="primary" pill>Primary Pill</bs-badge> <bs-badge variant="secondary" pill>Secondary Pill</bs-badge> <bs-badge variant="success" pill>Success Pill</bs-badge> </div>

Using the text Attribute

You can also set the badge content via the text attribute.

<div class="badge-container"> <bs-badge variant="primary" text="Text Attr"></bs-badge> <bs-badge variant="info" pill text="Pill with Text Attr"></bs-badge> </div>

Badges in Context

Badges scale to match the size of the parent element.
