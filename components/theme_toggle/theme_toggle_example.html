ThemeToggle Web Component Demo

Examples of the <theme-toggle> custom element.

1. Default Theme Toggle

This toggle uses the default theme localStorage key. It cycles through: Light → Dark → Auto (System).

<theme-toggle></theme-toggle>

2. Custom Storage Key

This toggle uses a custom localStorage key my-custom-theme. It operates independently of the others if they have different keys.

<theme-toggle storage-key="my-custom-theme"></theme-toggle>

3. Multiple Instances (Synced)

These two toggles use the same default key and stay in sync across the page.

<div class="d-flex gap-3"> <theme-toggle></theme-toggle> <theme-toggle></theme-toggle> </div>

4. Theme-aware Content

The content below changes its appearance based on the global theme set by the toggle.

Bootstrap Card

This card will automatically switch between light and dark modes because it uses Bootstrap's standard classes and the <theme-toggle> updates the data-bs-theme attribute on the <html> element.
