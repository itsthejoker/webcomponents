BsSpinner Web Component Demo

A custom web component wrapper for Bootstrap Spinners.

Border Spinner

The default spinner type is border.

<bs-spinner></bs-spinner>

Growing Spinner

Use the animation="grow" attribute for a growing spinner.

<bs-spinner animation="grow"></bs-spinner>

Colors

Use the variant attribute to apply contextual colors.

<!-- Border Spinners --> <bs-spinner variant="primary"></bs-spinner> <bs-spinner variant="secondary"></bs-spinner> <bs-spinner variant="success"></bs-spinner> <bs-spinner variant="danger"></bs-spinner> <bs-spinner variant="warning"></bs-spinner> <bs-spinner variant="info"></bs-spinner> <bs-spinner variant="light"></bs-spinner> <bs-spinner variant="dark"></bs-spinner> <!-- Growing Spinners --> <bs-spinner animation="grow" variant="primary"></bs-spinner> <bs-spinner animation="grow" variant="secondary"></bs-spinner> <bs-spinner animation="grow" variant="success"></bs-spinner> <bs-spinner animation="grow" variant="danger"></bs-spinner> <bs-spinner animation="grow" variant="warning"></bs-spinner> <bs-spinner animation="grow" variant="info"></bs-spinner> <bs-spinner animation="grow" variant="light"></bs-spinner> <bs-spinner animation="grow" variant="dark"></bs-spinner>

Size

Use the small attribute for a smaller spinner.

<bs-spinner small></bs-spinner> <bs-spinner animation="grow" small></bs-spinner>

Alignment

Spinners are display: inline-flex, so you can use utility classes for alignment.

Flexbox
Text Alignment
Floats
<!-- Flexbox --> <div class="d-flex justify-content-center"> <bs-spinner></bs-spinner> </div> <!-- Text Alignment --> <div class="text-center"> <bs-spinner variant="primary"></bs-spinner> </div> <!-- Floats --> <div class="clearfix"> <bs-spinner class="float-end"></bs-spinner> </div>

Buttons

Spinners can be used inside buttons.

<button class="btn btn-primary" type="button" disabled> <bs-spinner small label="Loading..." aria-hidden="true"></bs-spinner> <span class="visually-hidden">Loading...</span> </button> <button class="btn btn-primary" type="button" disabled> <bs-spinner small aria-hidden="true"></bs-spinner> Loading... </button>

Custom Label

Change the screen reader text with the label attribute.

<bs-spinner label="Please wait..."></bs-spinner>
